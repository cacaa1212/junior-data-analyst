Memulai Perjalanan sebagai Junior Data Analyst

Halo, perkenalkan nama saya Friska Cahya Ramadani. Saya adalah seorang Junior Data Analyst yang memiliki minat dalam pengolahan dan analisis data. Saya terbiasa bekerja secara terstruktur dan terdokumentasi, serta terus mengembangkan kemampuan analisis data melalui pembelajaran dan project sederhana.

Melalui Medium, saya ingin mendokumentasikan proses belajar, berbagi pengalaman, dan membangun portofolio di bidang data.

Dalam proses belajar, saya menggunakan beberapa tools utama. Google Drive saya gunakan untuk menyimpan dan mengelola data project agar tersusun rapi. Python saya gunakan sebagai bahasa pemrograman untuk membantu analisis data dan melatih logika pengolahan data. Selain itu, Spreadsheet (Google Sheets) saya manfaatkan untuk data cleaning, analisis dasar, dan visualisasi data.

Tujuan saya mempelajari data analyst adalah untuk mengembangkan kemampuan analisis data, melatih pola pikir logis, serta membangun portofolio sebagai Junior Data Analyst.

Terima kasih telah meluangkan waktu untuk membaca.
